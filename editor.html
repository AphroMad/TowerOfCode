Shortcuts
B Brush tool
E Eraser tool
G Ground layer
W Objects layer
F Effects layer
Esc Deselect
Ctrl+Z Undo
R-click Delete entity
Layers
Ground — Visual tiles the player sees
Objects — Collision tiles that block movement (rocks, invisible walls)
Effects — Behavioral tile effects (ice sliding, redirect arrows)
Entities — Click to select and edit spawns, NPCs, warps
Entity Markers
● S = Player spawn
● P = Professor
● G = Gatekeeper
● F = Flavor NPC
● W = Warp point
NPC Behaviors
Static — Player must interact manually
Detect — Line-of-sight, approaches player
Lookout — Rotates facing, detects player
Patrol — Walks waypoints, detects player
Shift+click = add patrol waypoint
Workflow
1. Paint ground tiles
2. Switch to Objects, paint collision
3. Place spawn, NPCs, warps
4. Set NPC behaviors
5. Export JSON + TS
